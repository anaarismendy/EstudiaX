El presente proyecto consiste en el desarrollo de una interfaz gráfica web interactiva que integra dos enfoques de inteligencia artificial: un Sistema Experto basado en reglas y un modelo de Lógica Difusa, orientados al análisis del rendimiento y bienestar estudiantil.

Por un lado, el sistema experto permitirá predecir el riesgo de que un estudiante pierda una materia, a partir de un conjunto de hechos académicos objetivos tales como: promedio actual, número de inasistencias, cantidad de horas de estudio y nivel de participación en clase. Mediante reglas de producción (IF–THEN), el agente realizará inferencias para clasificar el riesgo en niveles como bajo, medio o alto, ofreciendo una evaluación fundamentada en criterios académicos previamente definidos.

Por otro lado, el sistema incorporará un módulo de Lógica Difusa para evaluar el nivel de estrés del estudiante. Este modelo trabajará con variables como horas de sueño, carga académica y nivel de ansiedad, permitiendo manejar la incertidumbre y la gradualidad propia de factores emocionales. Como resultado, el sistema determinará un nivel de estrés categorizado como bajo, moderado o alto, utilizando grados de pertenencia y un proceso de defuzzificación.

De esta manera, la plataforma no solo analizará el desempeño académico desde una perspectiva lógica y estructurada, sino que también considerará variables humanas y subjetivas mediante técnicas de inteligencia artificial, proporcionando una evaluación integral del estado académico y emocional del estudiante.
